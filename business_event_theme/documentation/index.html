CSS Files and Structure

Compare the comments in CSS file with comments in HTML file for better understanding. Use main.css file for content, layout and format changes, this file separated in the sections by comments. You can easily navigate through it to find rules you wish to edit. Here's a list of the stylesheet files I'm using with this template, you can find more information opening each file:

File Name Path Description
bootstrap.css bootstrap/css/bootstrap.css Twitter Bootostrap framework file.
animate.css assets/css/plugins/animate/animate.css Cross-browser CSS animations.
bootstrap-select.css assets/css/plugins/select/bootstrap-select.css A custom select for Twitter Bootstrap using button dropdown
main.css assets/css/main.css General file with all template styles.

Reveal CSS Animations

  1. Make an element revealable

    Add the CSS class .wow to a HTML element: it will be invisible until the user scrolls to reveal it. (You can change this CSS class in the WOW settings to avoid name conflicts.)
